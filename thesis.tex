\begin{spodnitext}{Abstract}
%Práce se zabývá použitím protokolu \Zkratka{IPv6}{Internet Protocol version 6} na sítových prvcích Juniper s~operačním systémem \Zkratka{JUNOS}{Juniper Operating System}, zejména řadě SRX. Zaměřuje se na praktické problémy s~nasazením v~síti a problematikou přidělování \Zkratka{IP}{Internet Protocol} adres a síťových parametrů.\\
%\textbf{Klíčová slova:} \Zkr{IPv6}, \Zkr{JUNOS}, Juniper, \Zkratka{DHCPv6}{Dynamic Host Configuration Protocol version 6}, \Zkratka{SLAAC}{Stateless Address Autoconfiguration}
%
%\vspace{1cm}
	This master thesis deals with testing data networks in cloud environment. Techniques of inter and intra-cloud networks are described in theoretical part as well as virtual machine migrations.
	Practical part brings methodology and framework for testing virtual machine migration. Measurements are performed at OpenNebula cloud environment with {KVM} virtual machines.
\end{spodnitext}
